# Post Gal

Bay Area platform connecting artists and hosts. House shows, open studios, skill swaps, and community.

## Run

```
cd postgal
python3 server.py
```

Open http://localhost:8888

## Demo accounts

Any username from the People page with password `demo1234`. For example:
- `mara_luna` — muralist, Mission District
- `sage_house` — backyard host, Bernal Heights
- `devonte_k` — jazz/electronic, West Oakland

## Stack

- **Backend:** Python / Tornado / SQLite
- **Frontend:** Vanilla HTML/CSS/JS (single file)
- No external dependencies beyond what's in the Python stdlib + Tornado
